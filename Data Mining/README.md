# Data Mining and Machine Learning – Semester Projects

This folder contains laboratory exercises and case studies focused on data preprocessing, clustering, classification, customer segmentation, time-series analysis, and model evaluation.

## Contents

### `Forests.ipynb`

Machine learning experiments using decision trees, Random Forest, HistGradientBoosting, and k-nearest neighbours. The notebook includes model regularization, hyperparameter tuning, feature importance, SHAP explanations, fairness analysis, PCA, KD-trees, and Optuna optimization.

### `Ingredients_AGH_LAB.ipynb`

Clustering analysis of food products based on their nutritional characteristics. It covers missing-value handling, outlier treatment, scaling, dimensionality reduction, cluster evaluation, and comparison of methods such as K-Means, DBSCAN, hierarchical clustering, and Gaussian Mixture Models.

### `IRIS_AGH_LAB.ipynb`

An extended clustering analysis of the Iris dataset. The notebook compares K-Means, DBSCAN, Spectral Clustering, Kohonen Self-Organizing Maps, and other methods using internal and external evaluation metrics. It also examines cluster stability with bootstrap sampling.

### `MarketingAnalysis_AGH_LAB.ipynb`

Customer segmentation based on marketing campaign data. The analysis includes data cleaning, feature engineering, outlier handling, scaling, K-Means clustering, PCA visualization, and interpretation of customer segments.

### `Katarzyna_Wesolowska_CaseStudy.html`

A complete data science case study created in R. The project predicts the number of fatalities in road accidents and compares a Multi-Layer Perceptron neural network with a Random Forest regression model.

### `lab4.m`

MATLAB analysis of EUR and CHF exchange-rate time series. The script covers date alignment, linear interpolation, return and volatility calculations, linear and polynomial trends, residual analysis, and moving-average smoothing.
