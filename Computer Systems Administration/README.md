## **Computer Systems Administration**

This folder contains coursework completed as part of the *Computer Systems Administration* course, centered around deploying and managing a web application in a server environment.
The project involved containerizing a Python Flask application with Docker, building and publishing images to a container registry, and running the service on a remote server. The setup included network configuration, firewall rules, reverse proxy configuration with Nginx, DNS resolution, and secure access via OpenVPN.
The work focused on practical system administration tasks such as service deployment, networking, access control, and verifying system availability in a real server environment.
