# Student Projects Portfolio

A collection of university projects covering data analysis, machine learning, statistical modeling, spatial analysis, optimization, databases, and artificial intelligence.

## Repository Navigation

| Folder | Description |
|---|---|
| [Computational Intelligence in Data Analysis](./Computational%20Intelligence%20in%20Data%20Analysis/) | Genetic algorithms, fuzzy logic, Monte Carlo simulation, and neural networks. |
| [Data Analysis in Python](./Data%20Analysis%20in%20Python/) | Data cleaning, exploratory analysis, visualization, regression, clustering, and introductory geospatial analysis. |
| [Data Mining](./Data%20Mining/) | Clustering, customer segmentation, tree-based models, model interpretation, and time-series analysis. |
| [Engineering Thesis - Stroke Risk Prediction](./Engineering%20Thesis%20-%20Stroke%20Risk%20Prediction/) | Machine learning project for stroke-risk prediction using imbalanced medical data. |
| [Machine Learning](./Machine%20Learning/) | Classification, regression, clustering, feature engineering, and model evaluation. |
| [Others](./Others/) | Additional projects, including a homomorphic electronic voting system. |
| [SQL](./SQL/) | Relational databases, SQL queries, procedures, functions, and database design. |
| [Spatial Data Analysis](./Spatial%20Data%20Analysis/) | Spatial point processes, covariates, density estimation, DBSCAN, and HDBSCAN. |
| [Statistical Modeling](./Statistical%20Modeling/) | Linear, polynomial, Poisson, logistic, and ordinal regression with model diagnostics. |
| [Stochastic Methods in Machine Learning](./Stochastic%20Methods%20in%20Machine%20Learning/) | Gradient-based, Bayesian, evolutionary, and multiobjective optimization, plus Markov-chain analysis. |

## Technologies

Python, R, MATLAB, SQL, Jupyter Notebook, scikit-learn, XGBoost, TensorFlow, PyTorch, GeoPandas, PostGIS, Shiny, OpenGL, and related data-analysis tools.

Each folder contains its own `README.md` with a more detailed description, file structure, and instructions.
