# Computational Intelligence in Data Analysis

This folder contains projects completed for the **Computational Intelligence in Data Analysis** course. The notebooks present practical applications of genetic algorithms, fuzzy logic, Monte Carlo simulation, and neural networks.

## Projects

### Genetic Algorithm for the Traveling Salesman Problem

Implementation of a genetic algorithm for finding a short route between cities. The notebook includes population initialization, tournament selection, crossover, mutation, convergence analysis, and route visualization.

**File:** `01-genetic-algorithm-tsp.ipynb`

### Fuzzy Logic Fertilizer Recommendation

A prototype fuzzy inference system that recommends a fertilizer dose based on soil magnesium, phosphorus, potassium, and pH values. The project defines membership functions, a complete fuzzy rule base, and centroid defuzzification.

**Files:**

- `02-fuzzy-logic-fertilizer-recommendation.ipynb`
- `gleba.jpg`

> This project is an educational prototype and should not be treated as professional agricultural advice.

### Monte Carlo Estimation of Pi

Simulation-based estimation of the value of π for different sample sizes. The notebook visualizes randomly generated points, convergence toward π, and the reduction of estimation variability as the number of simulations increases.

**File:** `03-monte-carlo-pi-estimation.ipynb`

### Neural Network Function Approximation

A TensorFlow neural network trained to approximate a nonlinear mathematical function from noisy observations. The notebook explores network depth and width, training and validation error, underfitting, overfitting, early stopping, and learning-rate reduction.

**File:** `04-neural-network-function-approximation.ipynb`
