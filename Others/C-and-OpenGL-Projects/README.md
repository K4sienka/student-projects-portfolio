# C and OpenGL Projects

This folder contains university projects written in C and C++, including implementations of fundamental data structures, algorithms, numerical methods, console applications, and interactive 3D graphics developed with OpenGL.

## C and C++ Projects

### Binary Search Tree Word Indexer

A Binary Search Tree application that reads words from a text file, counts their occurrences, records the line numbers on which they appear, and outputs the results in alphabetical order.

### FIFO Queue

A FIFO queue implemented in C++ with a dynamically allocated linked structure. The project supports adding, removing, and displaying queue elements, together with basic exception handling.

### Linked List

A dynamically managed linked list in C supporting element insertion, deletion, searching, sorting, and complete memory cleanup.

### Matrix Operations

A collection of matrix operations in C, including recursive determinant calculation, transposition, inversion, and solving systems of linear equations using the inverse matrix method.

### Custom `printf`

A simplified implementation of the standard `printf` function. It supports integers, floating-point numbers, characters, and strings through custom output functions.

### Priority Queue

A dynamic priority queue implemented with a heap-based structure. It supports insertion, removal of the highest-priority element, priority updates, and queue visualization.

### Reverse Polish Notation Calculator

A calculator for mathematical expressions based on stacks. It supports arithmetic operators, exponentiation, operator handling, and division-by-zero validation.

### Ship Navigation Simulation

A recursive pathfinding simulation in which a ship searches for a valid route through a grid-based map while respecting water-depth constraints and avoiding previously visited cells.

### Sorting Algorithms Comparison

An implementation and performance comparison of several sorting algorithms:

- Bubble Sort
- Insertion Sort
- Selection Sort
- Mixed Sort
- Half Insertion Sort
- Heap Sort
- Quick Sort

The project measures execution time on arrays of different sizes and allows the sorted output to be verified.

## OpenGL Projects

### OpenGL 3D Graphics

An interactive 3D scene created with OpenGL and GLSL. The project includes textured objects, a dynamic coffee-shaped model, a rotating polyhedron, shader-based rendering, BMP texture loading, and keyboard and mouse camera controls.

### OpenGL 3D Graphics v2

A second version of the OpenGL project containing further work on the 3D scene, textures, shaders, object rendering, and interactive camera navigation.
